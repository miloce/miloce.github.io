大学生必领福利：GitHub 学生包申请详细步骤

发表于2025-03-16|更新于2026-03-16
|字数总计:1.3k|阅读时长:4分钟|评论数:

上周刚白嫖到 GitHub 学生包，JetBrains 全家桶免费用一年，Copilot 写代码爽到飞起！

今天就把我的申请经验分享给大家，全程 10 分钟搞定，手残党也能学会！

一. GitHub 学生包是什么？

说实话，我之前也不知道这玩意儿。

直到看到室友在用 IntelliJ IDEA 写 Java，界面酷炫的一批，一问才知道是 GitHub 学生包送的。

简单说，GitHub 给全球大学生发福利，只要你能证明自己是学生，就能免费领取价值好几万的开发工具：

GitHub Copilot —— AI 帮你写代码，自动补全神器

JetBrains 全家桶 —— IDEA、PyCharm、WebStorm 全免费

GitHub Pro —— 无限私有仓库

Azure $100 额度 —— 微软云服务器随便玩

免费域名 —— 搭个人博客必备

还有 50+ 工具…

📌 申请入口：https://github.com/education

二. 申请前的准备工作

很多人认证失败，问题往往出在准备阶段。下面这几件事 必须提前做好，不然大概率被拒。

2.1 确保你有学校邮箱

学校邮箱一般是 .edu 或者 .edu.cn 结尾，也有些学校用自己的域名（比如 @xxx.university.cn）。

如果你注册 GitHub 用的不是学校邮箱，需要先把学校邮箱添加进去：

  1. 打开 GitHub，点击右上角头像 → Settings
  2. 左侧找到 Emails
  3. 点击 Add email address，输入你的学校邮箱
  4. 去学校邮箱里点验证链接，完成验证

2.2 把个人资料改成真实信息

这一步很关键，GitHub 审核时会比对你提交的学生证明和账号信息。

  1. 打开 Settings → Public profile
  2. Name：改成和学生证一致的姓名（大写拼音）

另外建议顺便把 Billing and plans → Payment information 里的姓名和地址也补全，保持和学生证一致。（同样使用大写拼音）

2.3 配置好2FA

这一步是必须的，点击后来到下图：

这时候，需要下载一个Auhenticator，打开扫描这个二维码。扫描后，会发一个6位数码给你。如图：

然后输入验证码。就会来到下面界面：

你点击下载之后，下面绿色就就可以点击

2.4 准备好学生证明材料

审核时需要上传能证明你是在读学生的材料，优先推荐学生证，要求：

  1. 能看清 学校名称
  2. 能看清 你的姓名
  3. 能看清 有效期（入学日期、毕业日期或当前学期日期）

如果学生证不在手边，也可以用：

  • 成绩单
  • 课程表（带学期信息）
  • 学籍证明 / 在读证明
  • 学信网验证报告（登录 chsi.com.cn 申请）

三. 学生认证申请步骤

准备工作做完，下面正式开始申请。整个流程大概 5-10 分钟。

3.1 进入申请页面

打开：https://github.com/settings/education/benefits

或者从 GitHub 首页点：头像 → Settings → 左侧 Education → Education benefits

找到「GitHub Education」板块，点击 Start an application。

3.2 选择身份并填写信息

  1. 身份选 Student
  2. 学校名称会有下拉列表，输入你学校的名称，从列表里选。有Select this school选项可以直接选择!
  3. 点击Share Location共享位置，需要确保位置在学校内或附近，不然后面可能会多一个验证，不在学校的可以通过F12开发人员工具进行伪装位置。

3.3 上传学生证明

这一步是重点，需要用摄像头拍摄你的学生证（或其他证明材料）。

⚠️ 踩坑提醒

• 必须用真实摄像头拍摄，不能用虚拟摄像头或者直接上传图片文件• 拍摄时确保光线充足，学校名称、姓名、日期都要能看清• 用国内网络直连，不能使用学术上网• 定位权限要允许，最好在学校或学校附近操作

3.4 提交并等待审核

点击 Submit application 提交。

审核时间：

  • 大部分情况下是 机器审核，几分钟内就有结果
  • 少数情况需要人工复核，可能要 1-3 天

审核通过后，你会收到邮件通知，GitHub 账号也会显示学生身份标识。

四. 领取福利

学生认证通过后，福利不会自动开通，还需要手动领取。

4.1 进入 GitHub Student Developer Pack

打开到我们申请学生认证界面，有一个学生包链接和copilot pro会员领取的超链接我们可以。或直接打开：https://education.github.com/pack

4.2 领取学生包

页面上会显示所有可用的福利，点击 对应的领取链接 逐个领取：

后续我也会更新针对小白的免费服务器，域名等领取教程，感兴趣的可以关注微信公众号“开源栈”

五. 总结

整个流程梳理一下：

  1. 准备阶段： 添加学校邮箱、做好2FA认证、个人资料改真实信息、准备好学生证
  2. 申请认证： 进入 Education 页面 → 选 Student → 拍学生证 → 提交
  3. 领取福利： 认证通过后去 Student Pack 页面点击获取权限
